**命令**

	pip install virtualenv	//安装
	virtualenv testvir		//创建方法1
	cd ./Scripts			//进入目录
	activate.bat    //进入
	pip list		//列出
	deactivate.bat  //退出
	pip install virtualenvwrapper-win	//安装
	mkvirtualenv testvir2   //创建方法2
	deactivate				//退出
	workon					//进入
	workon testvir2			//进入指定
	pip install requests	//安装模块
